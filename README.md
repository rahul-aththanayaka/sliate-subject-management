# 📚 SLIATE Subject Management System

A streamlined web application developed as a partial fulfillment of the **HNDIT 1st Year Web Assignment**. This project focuses on managing academic subjects efficiently using a PHP and MySQL backend with a Bootstrap-powered frontend.

---

## 🚀 Key Features
- **Secure Login System:** Protects the management dashboard from unauthorized access.
- **Subject Management (CRUD):**
  - **Add New Subjects:** Easily input subject details into the system.
  - **Update/Edit:** Modify existing subject records.
  - **Delete:** Remove outdated or incorrect subject entries.
- **Search Functionality:** Quickly find specific subjects using the built-in search feature.
- **Responsive Design:** Built using **Bootstrap 5** to ensure the layout works on various screen sizes.

---

## 🛠️ Tech Stack
- **Backend:** PHP (Procedural)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript, **Bootstrap 5**
- **Server Environment:** XAMPP / WAMP

---

## 📋 How to Setup & Run

1. **Download/Clone** this repository to your local machine.
2. Move the project folder to your local server directory (e.g., `C:/xampp/htdocs/`).
3. Open **phpMyAdmin** (`localhost/phpmyadmin`).
4. Create a new database named **`webapp`**.
5. **Import** the provided **`webapp.sql`** file into the `webapp` database.
6. Run the project in your browser: `http://localhost/sliate-subject-management/`
